          • dbus._dbus.Bus: A connection to one of three possible standard buses, the SESSION, SYSTEM, or STARTER bus. This class manages shared connections to those buses.

            • dbus._dbus.StarterBus: The bus that activated this process (only valid if this process was launched by DBus activation).
            • dbus._dbus.SystemBus: The system-wide message bus.

    • dbus.service._VariantSignature: A fake method signature which, when iterated, yields an endless stream of 'v' characters representing variants (handy with zip()).
    • basestring: Type basestring cannot be instantiated; it is the base for str and unicode.
      • str: str(object) -> string
        • _dbus_bindings._StrBase: Base class for str subclasses with a variant_level attribute. Do not rely on the existence of this class outside dbus-python.
          • dbus.ByteArray: ByteArray is a subtype of str which can be used when you want an efficient immutable representation of a D-Bus byte array (signature 'ay').
          • dbus.ObjectPath: A D-Bus object path, such as '/com/example/MyApp/Documents/abc'.
          • dbus.Signature: A string subclass whose values are restricted to valid D-Bus signatures. When iterated over, instead of individual characters it produces Signature instances representing single complete types.
          • dbus.UTF8String: A string represented using UTF-8 - a subtype of str.
      • unicode: unicode(string [, encoding[, errors]]) -> object
        • dbus.String: A string represented using Unicode - a subtype of unicode.
    • dict: dict() -> new empty dictionary dict(mapping) -> new dictionary initialized from a mapping object's (key, value) pairs dict(iterable) -> new dictionary initialized as if via: d = {} for k, v in iterable: d[k] = v dict(**kwargs) -> new dictionary initialized with the name=value pairs in the keyword argument list.
      • dbus.Dictionary: An mapping whose keys are similar and whose values are similar, implemented as a subtype of dict.
    • float: float(x) -> floating point number
      • _dbus_bindings._FloatBase: Base class for float subclasses with a variant_level attribute. Do not rely on the existence of this class outside dbus-python.
        • dbus.Double: A double-precision floating point number (a subtype of float).
    • int: int(x[, base]) -> integer
      • _dbus_bindings._IntBase: Base class for int subclasses with a variant_level attribute. Do not rely on the existence of this class outside dbus-python.
        • dbus.Boolean: A boolean, represented as a subtype of int (not bool, because bool cannot be subclassed).
        • dbus.Byte: An unsigned byte: a subtype of int, with range restricted to [0, 255].
        • dbus.Int16: A signed 16-bit integer between -0x8000 and +0x7FFF, represented as a subtype of int.
        • dbus.Int32: A signed 32-bit integer between -0x8000 0000 and +0x7FFF FFFF, represented as a subtype of int.
        • dbus.UInt16: An unsigned 16-bit integer between 0 and 0xFFFF, represented as a subtype of int.
    • list: list() -> new empty list list(iterable) -> new list initialized from iterable's items
      • dbus.Array: An array of similar items, implemented as a subtype of list.
    • long: long(x[, base]) -> integer
      • _dbus_bindings._LongBase: Base class for long subclasses with a variant_level attribute. Do not rely on the existence of this class outside dbus-python.
        • dbus.Int64: A signed 64-bit integer between -0x8000 0000 0000 0000 and +0x7FFF FFFF FFFF FFFF, represented as a subtype of long.
        • dbus.UInt32: An unsigned 32-bit integer between 0 and 0xFFFF FFFF, represented as a subtype of long.
        • dbus.UInt64: An unsigned 64-bit integer between 0 and 0xFFFF FFFF FFFF FFFF, represented as a subtype of long.
    • tuple: tuple() -> empty tuple tuple(iterable) -> tuple initialized from iterable's items
      • dbus.Struct: An structure containing items of possibly distinct types.
    • type: type(object) -> the object's type type(name, bases, dict) -> a new type
      • gi._gobject.GObjectMeta: Metaclass for automatically registering GObject classes
        • dbus.gobject_service.ExportedGObjectType: A metaclass which inherits from both GObjectMeta and dbus.service.InterfaceType. Used as the metaclass for ExportedGObject.
      • dbus.service.InterfaceType
        • dbus.gobject_service.ExportedGObjectType: A metaclass which inherits from both GObjectMeta and dbus.service.InterfaceType. Used as the metaclass for ExportedGObject.
